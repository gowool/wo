# Wo - Modern HTTP Web Framework for Go

[![Go Reference](https://pkg.go.dev/badge/github.com/gowool/wo.svg)](https://pkg.go.dev/github.com/gowool/wo)
[![Go Report Card](https://goreportcard.com/badge/github.com/gowool/wo)](https://goreportcard.com/report/github.com/gowool/wo)
[![codecov](https://codecov.io/github/gowool/wo/graph/badge.svg?token=RQNG2GDDU6)](https://codecov.io/github/gowool/wo)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

**Wo** is a modern, high-performance HTTP web framework built with Go 1.25+ generics. It features a type-safe routing system with comprehensive middleware support, event-driven request processing, and a clean, expressive API.

### Installation

```bash
go get github.com/gowool/wo
```
